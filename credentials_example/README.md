# bsky-bots.queerlil.tools credentials

## <service_name>.json
This is the main credential file for the named service. It should contain an "identifier" key/value pair, which is the user's email, and a "password" k/v pair.

## <service>.session.json
This file is automatically generated for the specfiied service, and is used to persist a session and reduce the amount of authentication calls made.

## handles.remove.key
This is the password used by the bsky-handles.ps1 CGI script, to authenticate that removal requests are being made by the backend service
